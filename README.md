# cnsite-wanbo-guide

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为通用页面集合的存储与展示空间。

## 项目简介

本项目旨在整理和公开一系列独立 HTML 页面。这些页面可能涉及不同主题或用途，但均以纯前端静态页面形式存在。仓库本身不提供任何导航、索引或跳转服务，仅作为页面资源的归档点。

## 目录说明

```
cnsite-wanbo-guide/
├── pages/              # 存放所有独立 HTML 页面
├── assets/             # 页面所需的静态资源（如 CSS、JS、图片等）
├── index.html          # 仓库首页（可选，视情况而定）
└── README.md           # 本说明文件
```

- **pages/**：每个 HTML 页面为一个独立文件，文件名反映页面内容主题。
- **assets/**：按页面或通用需求组织的资源文件夹。
- **index.html**：如存在，仅作为仓库根目录的占位或简单引导页面。

## 页面归档说明

- 所有页面均为纯前端实现，不依赖后端服务。
- 页面内容可能随时间更新或调整，历史版本可通过 Git 提交记录追溯。
- 每个页面文件独立存在，彼此之间无强制关联。
- 页面中不包含外链跳转或第三方服务调用，除非在页面内明确标注。

## 维护说明

- 本仓库由维护者不定期更新，添加或修改页面。
- 欢迎通过 Issues 提交建议或报告问题，但请勿提交涉及具体域名、网站或营销内容的请求。
- 不接受涉及商业推广、诱导点击或违反平台政策的页面提交。
- 所有页面内容应符合开源社区基本规范，不得包含违法或侵权信息。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件即可查看对应页面。
3. 如需部署，可将整个仓库内容上传至任意静态托管服务。
